# ASDI Tables

Structured sustainability and open-data dictionaries used as inputs for the
policy-graph experiments.

## Contents

- `01 ASDI Data Sets Dictionary.xlsx`: working spreadsheet dictionary.
- `02 ASDI Data Set Dictionary.csv`: CSV version of the ASDI dataset dictionary.
- `GBI Data Set/GBI MX Dataset.csv`: Mexico-focused biodiversity occurrence
  data used in exploratory graph extraction.

## Provenance Status

The files are thesis-supporting research inputs. Some rows include source-like
fields, licenses, and dataset names, but formal publication requires explicit
source URLs, access dates, and checksums for every data table.

See `../docs/provenance.md` for the repository-level provenance register.
