# Public Policies Tables

Structured policy and development-plan tables used for graph and recommendation
experiments in the doctoral thesis work.

## Contents

- `01 Tabla B1.csv`: seed CSV with SDG indicator text and classification labels.
- `01 Tabla B1.xlsx` through `49 Tabla 21 .xlsx`: spreadsheet tables used as
  working inputs for policy graph construction and analysis.

## Provenance Status

These files are thesis-supporting research inputs. Before formal publication,
each table needs a table-level source note with:

1. source institution or source document;
2. source URL or archival identifier;
3. extraction or access date;
4. transformation steps;
5. license or permitted-use basis;
6. relationship to thesis chapter, figure, table, or appendix.

See `../docs/provenance.md` for the repository-level provenance register.
