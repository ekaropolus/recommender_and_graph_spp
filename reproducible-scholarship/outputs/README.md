# Generated Outputs

These outputs are generated by:

```bash
python scripts/build_reproducible_scholarship.py
```

They are committed so reviewers can inspect the reproducible graph slice without
running the build first.

## Contents

- `tables/file_inventory.csv`: source file paths, roles, sizes, and SHA-256
  checksums.
- `tables/asdi_datasets.csv`: normalized ASDI dataset dictionary.
- `tables/sdg_policy_indicators.csv`: normalized SDG policy indicator table.
- `graph/nodes.csv`: portable graph nodes.
- `graph/edges.csv`: portable graph relationships.
- `manifests/summary.json`: generated row and graph counts.
- `manifests/run-manifest.json`: input inventory, output checksums, and known
  limitations for the reproducible run.

Validate with:

```bash
python scripts/test_reproducible_scholarship.py
```
